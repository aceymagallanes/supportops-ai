# Contributing

SupportOps AI is currently a personal portfolio project by Acey Magallanes.

The project is open for review, learning, and improvement ideas.

## Project Direction

The roadmap is focused on:

- Supabase persistence
- FastAPI backend
- OpenAI structured triage output
- Knowledge base semantic search
- n8n workflow automation
- Real support system integrations

## Local Development

Run the app locally:

```bash
python3 -m http.server 4173 --bind 127.0.0.1
```

Open:

```text
http://127.0.0.1:4173/index.html
```

## Contribution Guidelines

If contributing:

- Keep the app practical and operations-focused.
- Avoid adding generic AI hype.
- Do not commit real customer data.
- Do not commit API keys or secrets.
- Keep UI changes responsive.
- Document meaningful architecture or workflow changes.

## Suggested Issue Types

- Bug report
- UI improvement
- Documentation improvement
- Backend roadmap item
- Automation workflow idea
- Data model improvement
